# Glossary

> このファイルは、初学者がつまずきやすい用語を整理するためのテンプレートです。  
> 新しい用語、概念、略語が出てきたら、必要に応じて追記してください。

## 使い方

この用語集では、難しい言葉をできるだけやさしく説明します。  
厳密な定義を長く書くよりも、「まずはどう理解するとよいか」が伝わる形を優先するのがおすすめです。

## 書き方の目安

用語を追加するときは、次のような観点でそろえると読みやすくなります。

- 用語を一言で説明する
- どのチュートリアルや場面で出てくるかを書く
- 似た用語との違いがあれば短く補足する
- 初学者が最初に理解すべき範囲に絞る
