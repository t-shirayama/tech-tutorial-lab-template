# minimal-app

このフォルダは、複数のチュートリアルを組み合わせた最小構成のサンプルを置くための場所です。

テンプレート利用後は、以下のような内容を追加してください。

- 何を確認できるサンプルなのか
- どのチュートリアル内容を組み合わせているか
- 実行方法
- 確認ポイント

例:

- `tutorials/00-getting-started/` の環境を使って起動できる
- `tutorials/01-core-concepts/` の基本概念を小さなアプリで確認できる

必要であれば、このフォルダに `README.md`、コード、設定ファイルを追加して使ってください。
